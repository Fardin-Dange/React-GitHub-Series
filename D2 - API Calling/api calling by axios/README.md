# D2 - API Calling in React

## Topics Covered

- fetch() — built-in JS method to call APIs
- axios — third party library, better than fetch
- HTTP Methods — GET, POST, PUT, PATCH, DELETE
- API calls in separate file (userApi.js)
- CORS — what it is and how to fix it using WebMvcConfigurer
- 3 states in API call — data, loading, error

## APIs Used
- https://jsonplaceholder.typicode.com/users
- Spring Boot localhost:8080
